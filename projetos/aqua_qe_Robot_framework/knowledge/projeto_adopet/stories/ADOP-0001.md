## Objetivo

Como usuário interessado em adotar um animal de estimação, quero acessar a página inicial do AdoPet e visualizar as opções de navegação (ver pets, cadastrar ou fazer login), para que eu possa escolher a melhor forma de interagir com a plataforma e iniciar o processo de adoção.

## Regras de negócio

- RN01: A landing page é pública — acessível sem autenticação prévia.
- RN02: Botão "Ver pets disponíveis para adoção" redireciona para a vitrine/lista de animais.
- RN02: Link "Cadastrar" redireciona para o formulário de criação de conta.
- RN02: Link "Fazer login" redireciona para a tela de autenticação.
- RN03: Layout responsivo para mobile, tablet e desktop.

## Critérios de aceite mapeados para automação

- CA1: Página carrega com logotipo, ícones de navegação (Home e Mensagens) no cabeçalho.
- CA2: Texto de boas-vindas exibido: "Boas-vindas! Adotar pode mudar uma vida. Que tal buscar seu novo melhor amigo hoje? Vem com a gente!".
- CA3: Botão "Ver pets disponíveis para adoção" está visível e centralizado.
- CA4: Links "Cadastrar" e "Fazer login" estão visíveis.
- CA5: Ilustração dos animais (gato e cachorro) exibida na parte inferior.
- CA6: Clique em "Ver pets disponíveis para adoção" redireciona para a listagem de animais.
- CA7: Clique em "Cadastrar" redireciona para o formulário de registro.
- CA8: Clique em "Fazer login" redireciona para a tela de autenticação.

## Regras novas em vigor

- R1: Landing page pública sem exigência de login.
- R2: Três fluxos de navegação distintos: ver pets, cadastrar, fazer login.
- R3: Layout responsivo obrigatório.
